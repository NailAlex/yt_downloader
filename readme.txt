Для запуска бота потребуется следующее ПО:

Для Linux/Windows:
установленный пакет ffmpeg   (конвертация аудио во flac/mp3)
Установленный python3.9+ (тестировался именно на 3.9.2 на Raspberry4)

Установленный pip3, которым должны быть установлены библиотеки:
yt-dlp    (версии 2024.4.9+)
aiogram (версии 3+ все его зависимости подтянутся)
requests (версии 2.30.0+)
urllib3 (версии 2.2.0+)

Также для работы бота крайне рекомендуется сделать отдельный аккаут Google с почтой GMail и активацией в Youtube. От его имени(залогиниться) в Сhrome и открыть любое видео на Ютубе с 
пометкой "не для детей"(обычно все что не для киндеров с этой пометкой сейчас). Далее нужно выгрузить куки-файл c помощью плагина "Get cookies.txt LOCALLY 0.5.3" для Хрома в формате .txt 
и разместить его в "./src". Без данного куки-файла невозможно скачивать большинство файлов в Ютубе, так как пометки в видео требуют авторизованности скачивателя. Также ограничивается возможности
по разрешению и формату.
